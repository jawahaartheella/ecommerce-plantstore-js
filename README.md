# 🌿 GreenLeaf - Plant E-Commerce Platform

A modern, responsive e-commerce web application built using **Vanilla JavaScript, HTML5, and CSS3**. GreenLeaf provides a complete plant shopping experience, allowing users to browse products, search, filter, manage their cart, and complete the checkout process through an intuitive and responsive interface.

## 🎯 Project Purpose

The primary goal of this project was to strengthen core frontend development skills by building a complete e-commerce application **without using any frameworks or libraries**. The project focuses on understanding how modern web applications work behind the scenes by implementing features using pure JavaScript and browser APIs.
